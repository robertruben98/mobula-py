# mobula-py

Python client for the Mobula API (multi-chain crypto market & on-chain data). Sync + async, Pydantic v2.

## Install

```bash
pip install mobula-py
```

The distribution is `mobula-py`; the import package is `mobula`.

## Quick start

```python
import mobula
```

See the [API reference](api.md) for the full surface.
